# Luxe Japan Elite Escorts Website

一个奢华、高质感的静态多语言网站项目。

## 项目特点

- 🌐 多语言支持（繁体中文、英文、韩文、越南文、西班牙文）
- 📱 响应式设计
- 🎨 奢华现代UI设计
- 🔍 SEO优化
- 📊 集成分析工具
- 🧪 自动化测试

## 快速开始

### 环境要求

- Node.js 18+
- npm 9+

### 安装

```bash
# 克隆项目
git clone [项目地址]

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建项目
npm run build

# 预览构建结果
npm run preview
```

## 项目结构

```
src/
├── assets/          # 静态资源
├── i18n/           # 语言包
├── components/     # 组件
├── pages/         # 页面
└── styles/        # 样式
```
