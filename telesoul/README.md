# Telesoul

Telesoul is a social networking platform built with Next.js 14, focusing on meaningful connections and interactions.

## Features

- Modern UI with dark/light mode support
- Real-time chat functionality
- User profile management
- Wallet system with coins and diamonds
- Social media integration
- Voice call scheduling
- MBTI personality test
- Daily tasks and rewards

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Shadcn UI
- Lucide Icons
- Next-themes

## Getting Started

### Prerequisites

- Node.js 18.17 or later
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/telesoul.git
cd telesoul
```

2. Install dependencies
```bash
npm install
# or
yarn install
```

3. Create a `.env.local` file in the root directory
```bash
# Add your environment variables here
NEXT_PUBLIC_API_URL=your_api_url
```

4. Start the development server
```bash
npm run dev
# or
yarn dev
```

The application will be available at `http://localhost:3001`

### Deployment

#### Vercel (Recommended)

1. Push your code to GitHub
2. Import your repository in Vercel
3. Add your environment variables in Vercel's project settings
4. Deploy!

#### Other Platforms

1. Build the application
```bash
npm run build
# or
yarn build
```

2. Start the production server
```bash
npm run start
# or
yarn start
```
